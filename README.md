# İtfaiye
Arduino Uno ile; RFID Okuyucu, Gaz Sensörü, Alev Sensörleri, Park Sensörü, Otomatik Far yakma için Işık Sensörü gibi donanımları olan itfaiye aracı projesi.
Gorsel 1 İçeriğindeki Harfler:
A: LED
B: HC-SR04 Ultrasonik Mesafe Sensörü
C: Alev Sensörü
D: Buzzer
E: RC522 RFID Okuyucu
F: 0.96" OLED
G: 18650 Batarya
H: On/Off Switch
J: Mini Breadboard ve Pertinaks
K: LDR
